Box-sizing

estructura css: selector { propiedad: valor; }

box-sizing es una propiedad CSS que determina cómo se calcula el tamaño total de un elemento HTML. Esta propiedad controla si el ancho y el alto de un elemento incluyen o no su borde, relleno y contenido. Hay dos valores posibles para la propiedad box-sizing: content-box: es el valor por defecto. El ancho y alto de un elemento se calculan solo a partir del contenido, sin incluir su borde o relleno. Esto significa que si estableces un ancho de 100px en un elemento, ese ancho se aplicará solo al contenido, y el borde y el relleno se añadirán a ese ancho, aumentando el tamaño total del elemento. border-box: en este caso, el ancho y alto de un elemento incluyen su borde y relleno. Esto significa que si estableces un ancho de 100px en un elemento con un borde de 2px y un relleno de 10px, el ancho total del elemento será de 100px, incluyendo el borde y el relleno. La propiedad box-sizing es útil porque te permite controlar cómo se calcula el tamaño total de un elemento, lo que puede ser especialmente importante cuando trabajas con diseños responsivos o con elementos que deben ajustarse a un tamaño específico. Por ejemplo, puedes usar border-box para hacer que los elementos se ajusten al ancho de su contenedor sin tener que preocuparte por los tamaños de borde y relleno. La propiedad box-sizing afecta al elemento al que se le aplica la propiedad y a sus elementos internos. Si estableces la propiedad box-sizing en un elemento HTML, se aplicará a ese elemento y a todos sus elementos internos. Esto significa que el ancho y el alto de los elementos internos se calcularán de acuerdo con el valor de box-sizing establecido en su elemento padre.
