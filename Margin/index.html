MARGIN

La propiedad CSS "margin" define el espacio que se encuentra alrededor de un elemento HTML. Los valores se aplican en el siguiente orden: arriba, derecha, abajo, izquierda. En la sintaxis "margin 0px 0px 0px 0px", los cuatro valores especificados son iguales a cero, lo que significa que no se agregará ningún espacio alrededor del elemento. Los márgenes pueden utilizarse para separar los elementos entre sí y también para crear un espacio en blanco adicional en torno a los elementos, lo que puede ayudar a mejorar la legibilidad y el diseño de una página web. Por ejemplo, al establecer un margen izquierdo y derecho en "auto", se puede hacer que un elemento se centre en la página. Los márgenes también pueden ser útiles para ajustar el diseño en diferentes dispositivos y tamaños de pantalla. "margin" se aplica al elemento en el que se define. Es decir, el margen se aplica al espacio que rodea directamente al elemento en cuestión, no a los elementos que contiene. Por ejemplo, si se define un margen para un elemento "div", este margen se aplicará al espacio que rodea al elemento "div" en sí mismo, no a los elementos que se encuentran dentro del "div".
