Max-witdh and height

"max-width" es un término que se utiliza en diseño web y se refiere a la propiedad CSS que se utiliza para establecer la anchura máxima de un elemento en una página web. Es decir, esta propiedad define la anchura máxima que puede tener un elemento en la página web, y si su tamaño supera este límite, se ajustará automáticamente para mantenerse dentro de la anchura máxima establecida. La propiedad "max-width" es útil para garantizar que el contenido se ajuste bien en pantallas de diferentes tamaños, evitando que el diseño de la página se rompa o que el contenido se desborde fuera de la pantalla. Por ejemplo, si estableces una anchura máxima de 800 píxeles para una imagen en tu sitio web, la imagen se ajustará automáticamente si se muestra en una pantalla más pequeña, como la de un dispositivo móvil, para que encaje perfectamente en la pantalla sin distorsionarse o perder calidad. La propiedad CSS "max-width" se aplica al elemento al que se le está asignando la propiedad, así como a sus elementos hijos que se encuentran dentro de ese elemento. Por ejemplo, si se establece un ancho máximo de 800 píxeles para un contenedor que contiene varias imágenes, todas las imágenes dentro de ese contenedor se ajustarán automáticamente para adaptarse a ese ancho máximo de 800 píxeles. Esto se debe a que la propiedad "max-width" se aplica a todos los elementos que se encuentran dentro del contenedor que tiene esa propiedad establecida. Por lo tanto, la propiedad "max-width" puede afectar tanto al elemento al que se le asigna la propiedad como a sus elementos hijos. Es importante tener en cuenta esto al utilizar la propiedad "max-width" en tu diseño web para asegurarte de que el contenido se ajuste bien en todas las pantallas sin afectar negativamente al diseño general de tu sitio web.

Explicacion de porqué no se amplia el hijo a 1200 si el padre esta a 900

La propiedad "max-width" se refiere al ancho máximo que puede tener un elemento en una página web, mientras que la propiedad "max-height" se refiere a la altura máxima que puede tener el elemento. En el ejemplo que has proporcionado, has establecido una propiedad "max-width" de 1200px para el elemento con la clase "section". Sin embargo, también has establecido una propiedad "max-width" de 900px para el elemento padre con la clase "container". Dado que el elemento "section" es un elemento hijo del elemento "container", la propiedad "max-width" del elemento padre se está aplicando al elemento hijo "section" y está limitando su ancho máximo a 900px, en lugar de permitirle alcanzar su valor máximo de 1200px. Para solucionar este problema, debes asegurarte de que la propiedad "max-width" del elemento padre sea mayor o igual que la propiedad "max-width" del elemento hijo
