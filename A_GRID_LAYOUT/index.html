GRID LAYOUT


Teoría


Toma la filosofía y los valores de flexbox pero va más allá de sus limitaciones.
ya que flexbox es de una sola dimensión y grid es de dos dimensiones.
Conceptos:
- Grid container: El contenedor de todos los items
- Grid items: Es cada uno de los elementos del grid container. (Filas y columnas)
-Grid cel: Es cada uno de los elementos del grid contaiener (parte interna)
-Grid area: Es la parte interna de un conjunto de items
- Grid track: Es el conjunto de items pero en vertical y horizontal.
-Grid line: Cada una de las líneas (column line o row line)
Propiedades del elemento contenedor:
- grid-template-columns
- grid-template-rows
- justify-content
-align-items
-align-content
-grid-gap
-grid-template-areas

Propiedades de los items, los elementos hijos:
-grid-column
-grid-row
-grid-area
