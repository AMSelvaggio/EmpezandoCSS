Teoria Flex-Flow.

Estructura css: flex-flow{ propiedad: valor }

El flex-flow en CSS es una propiedad que combina las propiedades flex-direction y flex-wrap en una sola declaración. flex-direction establece la dirección del flujo de los elementos en un contenedor flex, mientras que flex-wrap especifica si los elementos deben envolverse o no cuando el espacio en el contenedor flex es insuficiente.

La sintaxis de flex-flow es la siguiente:

CSS Copy code flex-flow: < flex-direction> < flex-wrap>;

Donde < flex-direction> especifica la dirección del flujo de elementos en el contenedor flex y puede ser uno de los siguientes valores:

  • row: los elementos se colocan en una fila, de izquierda a derecha.
  • row-reverse: los elementos se colocan en una fila, de derecha a izquierda.
  • column: los elementos se colocan en una columna, de arriba a abajo.
  • column-reverse: los elementos se colocan en una columna, de abajo a arriba.

    • Por otro lado, < flex-wrap> especifica si los elementos deben envolverse o no cuando el espacio en el contenedor flex es insuficiente.

    Puede tener los siguientes valores:

  • nowrap: los elementos no se envuelven y se ajustan en una sola línea.
  • wrap: los elementos se envuelven y se muestran en varias líneas según sea necesario.
  • wrap-reverse: los elementos se envuelven y se muestran en varias líneas, pero en orden inverso.
