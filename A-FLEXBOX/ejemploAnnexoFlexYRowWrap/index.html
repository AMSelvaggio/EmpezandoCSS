Valor ROW WRAP y imagina el column wrap"

La propiedad "row wrap" se utiliza cuando se quiere que los elementos dentro de un contenedor se ajusten automáticamente en filas y que si no caben en una fila, se muevan automáticamente a la siguiente fila. Esto puede ser útil cuando se trabaja con elementos que tienen anchuras variables y se desea que se ajusten automáticamente dentro de un contenedor sin necesidad de establecer tamaños fijos. La propiedad "column wrap" se utiliza en combinación con la propiedad "display:flex" para crear diseños flexibles y adaptables a diferentes tamaños de pantalla, permitiendo que los elementos dentro de un contenedor se ajusten automáticamente en columnas y que si no caben en una columna, se muevan automáticamente a la siguiente columna.
