Display & FLEX

TEORIA

estructura css: selector { propiedad: valor; }

0 "Display" es una propiedad CSS que se utiliza para controlar cómo se muestra un elemento en una página web. Esta propiedad especifica el tipo de caja de visualización que se utilizará para el elemento y cómo se posicionará y se mostrará en relación con los demás elementos en la página. Existen diferentes valores que se pueden asignar a la propiedad "display", cada uno con su propia funcionalidad. Algunos de los valores más comunes son: "block": Este valor hace que el elemento se muestre como un bloque de ancho completo y se coloca debajo del elemento anterior. Los elementos de tipo "block" tienen un ancho, un alto, un margen y un relleno que se pueden ajustar. "inline": Este valor hace que el elemento se muestre en línea con el contenido que lo rodea y solo ocupa el ancho que necesita. Los elementos de tipo "inline" no tienen un ancho, un alto, un margen o un relleno que se puedan ajustar. SE aplica a parrafos y titulos (texto) "inline-block": Este valor combina las características de los elementos de tipo "block" y los elementos de tipo "inline", permitiendo que el elemento tenga un ancho, un alto, un margen y un relleno que se puedan ajustar, pero que se muestre en línea con el contenido que lo rodea. "none": Este valor hace que el elemento no se muestre en absoluto en la página web, es decir, no ocupa espacio y no es visible. La elección del valor correcto para la propiedad "display" dependerá del diseño y la funcionalidad que se quiera lograr en tu página web.

TEORIA: FLEX

"Display: flex" es un valor de la propiedad CSS "display" que se utiliza para crear diseños flexibles y dinámicos en una página web. Esta propiedad permite establecer un elemento como un "contenedor flexible" que puede alinear, distribuir y ajustar los elementos secundarios de manera más eficiente que otros métodos de diseño. Al establecer "display: flex" en un elemento contenedor, se crea un "flex container" que es responsable de distribuir y alinear los elementos secundarios dentro de él. Los elementos secundarios dentro del contenedor flexible se conocen como "flex items". La propiedad "display: flex" activa un modelo de diseño flexbox, que ofrece varias funcionalidades útiles para el diseño web, como la capacidad de ajustar dinámicamente el tamaño y la posición de los elementos secundarios, el centrado de elementos y la distribución uniforme de espacio entre ellos. A través de la combinación de propiedades como "justify-content" y "align-items", se pueden controlar el posicionamiento y la distribución de los elementos secundarios dentro del contenedor flexible. Con "display: flex", se puede crear diseños más responsivos y adaptables a diferentes dispositivos y tamaños de pantalla. En resumen, "display: flex" es una propiedad útil para el diseño web que permite crear diseños flexibles y dinámicos con menos código y mayor facilidad. Activa el flex. "flex" es una abreviatura de varias propiedades CSS que se utilizan en conjunto para establecer el modelo de diseño flexbox en un elemento contenedor. Las propiedades que se utilizan con mayor frecuencia junto con "flex" son las siguientes: "display": Como se mencionó anteriormente, "display: flex" establece el elemento contenedor como un contenedor flexible, lo que activa el modelo de diseño flexbox. "flex-direction": Esta propiedad establece la dirección del eje principal del contenedor flexible, lo que afecta a la dirección en la que los elementos secundarios se distribuyen dentro del contenedor. Los valores comunes para "flex-direction" son "row" (para distribuir los elementos secundarios horizontalmente), "column" (para distribuir los elementos secundarios verticalmente), "row-reverse" (para distribuir los elementos secundarios horizontalmente en sentido inverso) y "column-reverse" (para distribuir los elementos secundarios verticalmente en sentido inverso). "justify-content": Esta propiedad establece la forma en que los elementos secundarios se distribuyen en el eje principal del contenedor flexible. Los valores comunes para "justify-content" incluyen "flex-start" (para alinear los elementos secundarios en el extremo principal del contenedor), "flex-end" (para alinear los elementos secundarios en el extremo opuesto del contenedor), "center" (para alinear los elementos secundarios en el centro del contenedor), "space-between" (para distribuir uniformemente el espacio entre los elementos secundarios) y "space-around" (para distribuir uniformemente el espacio alrededor de los elementos secundarios). "align-items": Esta propiedad establece la forma en que los elementos secundarios se alinean en el eje transversal del contenedor flexible. Los valores comunes para "align-items" incluyen "flex-start" (para alinear los elementos secundarios en el extremo superior del contenedor), "flex-end" (para alinear los elementos secundarios en el extremo inferior del contenedor), "center" (para alinear los elementos secundarios en el centro del contenedor) y "stretch" (para hacer que los elementos secundarios se estiren para ocupar todo el espacio disponible en el eje transversal). "flex-wrap": Esta propiedad establece si los elementos secundarios deben envolverse o no en varias líneas dentro del contenedor flexible si no hay suficiente espacio disponible. Los valores comunes para "flex-wrap" incluyen "nowrap" (para hacer que los elementos secundarios se ajusten en una sola línea), "wrap" (para hacer que los elementos secundarios se ajusten en varias líneas) y "wrap-reverse" (para hacer que los elementos secundarios se ajusten en varias líneas en sentido inverso). Estas son solo algunas de las propiedades que se pueden utilizar en conjunto con "flex" para crear diseños flexibles y dinámicos en una página web. Hay muchas otras propiedades de flexbox disponibles que se pueden utilizar para ajustar el modelo de diseño y lograr diferentes resultados de diseño.

TEORIA DE PADRE A HIJO

La propiedad "display" afecta al elemento en el que se escribe y a sus hijos y también a los hijos de los hijos y así sucesivamente, es decir, a todos los elementos secundarios anidados dentro del elemento contenedor. Por ejemplo, si se establece "display: flex" en un elemento contenedor, todos los elementos secundarios dentro de ese contenedor se convierten en elementos flexibles y se pueden controlar utilizando las propiedades de flexbox. Si se establece "display: none" en un elemento contenedor, se oculta ese elemento y también se ocultan todos los elementos secundarios anidados dentro de él. Es importante tener en cuenta que las propiedades de CSS son heredadas de los elementos padres a los elementos hijos, lo que significa que las propiedades definidas en un elemento contenedor se aplican a todos los elementos secundarios anidados dentro de él, a menos que se especifiquen propiedades diferentes en esos elementos secundarios de manera específica. Por lo tanto, "display" afecta a todos los elementos secundarios dentro del elemento contenedor y su efecto se propaga a través de la jerarquía de elementos descendientes.
